About

Project Overview

AwareNest is a comprehensive User Experience Design (UED) project that focuses on creating intuitive and engaging interfaces. This interactive Figma prototype showcases the complete design system, user flows, and interface components developed for the project.

Key Features

• Interactive prototypes with realistic user flows
• Comprehensive design system and component library
• Responsive layouts for multiple device sizes
• Carefully crafted user experience patterns
• Modern visual design with attention to accessibility

How to Explore

Navigate through the design using the Figma embed above. You can zoom in/out, pan across different screens, and explore the complete design system. Use the controls within the Figma viewer to switch between different pages and view detailed component specifications.
