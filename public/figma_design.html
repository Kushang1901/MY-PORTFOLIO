About the Project

01 — Overview

Project Overview

AwareNest is a comprehensive User Experience Design (UED) project focused on creating intuitive and engaging interfaces for mental wellness. This interactive Figma prototype showcases the complete design system, user flows, and interface components developed for the project.

02 — Features

Key Features

Interactive prototypes with realistic user flows · Comprehensive design system and component library · Responsive layouts for multiple device sizes · Carefully crafted user experience patterns · Modern visual design with accessibility in mind.
